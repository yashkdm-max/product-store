Sportswear Product Management System
A full-stack MERN web application for managing and browsing sportswear products.
The application allows users to view products, search items, filter categories, and save favorite products to a wishlist.
This project demonstrates modern full-stack web development using React, Node.js, Express, and MongoDB.
Project Overview
The Sportswear Product Management System is designed to manage and display sportswear products such as shoes, t-shirts, hoodies, and other athletic clothing.
Users can explore available products, search for specific items, filter products by category, and add items to their personal wishlist.
The application also supports CRUD operations for product management including creating, updating, and deleting products.

Features
Product Management
The system allows full CRUD operations on products:
Create new products
View all products
Update product details
Delete products
Search Functionality
Users can search products by product name, making it easier to quickly find specific items.
Filter Products
Products can be filtered by category or gender, allowing users to narrow down the results based on their preferences.
Wishlist Feature
Users can add products to a wishlist using the heart icon.
The wishlist stores the selected products locally in the browser using localStorage.
Modern UI Design
The application uses Tailwind CSS to provide a clean and responsive user interface with a modern sportswear theme.
